Amenazas Comunes en Internet

El Phishing es un ataque en el que un atacante engaña a la víctima para que entregue información sensible, como contraseñas, datos bancarios o códigos de verificación.

  • Cómo funciona: Correos, mensajes o llamadas que parecen legítimos para engañar al usuario.
  • Ejemplos: Correos de bancos falsos, mensajes de "tu cuenta será suspendida", links que llevan a páginas idénticas a las reales.

El Ransomware es un tipo de malware que secuestra los archivos del usuario cifrándolos y luego solicita un rescate para devolver el acceso.

  • Cómo funciona: Archivos adjuntos, descargas o vulnerabilidades que instalan el malware.
  • Consecuencias: Pérdida de datos, interrupción de servicios, costos millonarios en recuperación.

Malware es cualquier software malicioso diseñado para causar daño o robar información.

  • Tipos: Virus, troyanos, spyware, adware.
  • Cómo se propaga: Descargas inseguras, sitios web maliciosos, dispositivos USB infectados.

Los ataques DDoS buscan colapsar un servidor o servicio enviando enormes cantidades de tráfico desde múltiples dispositivos.

  • Qué provocan: Sitio web o servicio no funciona, pérdidas económicas, interrupción de operaciones.
  • Cómo se ejecutan: Uso de botnets que envían millones de solicitudes simultáneas.
