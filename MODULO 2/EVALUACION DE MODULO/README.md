# Proyecto Web de Ciberseguridad

## Descripción
Este proyecto es un sitio web educativo sobre **ciberseguridad**.  
Permite conocer amenazas comunes, consejos de seguridad y realizar un pequeño test interactivo.


## Estructura del proyecto
MODULO 2/EVALUACION DE MODULO
- index.html # Página principal con introducción a la ciberseguridad
- amenazas.html # Página con las amenazas comunes (Phishing, Ransomware, Malware, DDoS)
- consejos.html # Página con consejos de seguridad y formulario de contacto
- css/styles.css # Archivo de estilos personalizados
- js/script.js # Scripts de interactividad, modal y test
- img/ # Carpeta con imágenes utilizadas en las páginas
- README.md # Documentación del proyecto


## Funcionalidades

- Navegación entre secciones.  
- Acordeón con información sobre Phishing, Ransomware, Malware y ataques DDoS.  
- Modal con test de seguridad básico.  
- Formulario de contacto.  
- Diseño responsive para móviles y escritorio.
